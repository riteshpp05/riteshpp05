## Projects

### Multimodal RAG Chatbot — Ollama · LangChain · LangGraph · FAISS · SQLite · OCR

A production-ready **offline Multimodal RAG system** built for real-world deployment with zero cloud dependency.

- Built semantic retrieval pipeline using **Ollama (llama2:7b)**, `nomic-embed-text` embeddings, and **FAISS** for accurate document search across PDFs and OCR-extracted data
- Implemented **persistent conversational memory** using LangGraph + SQLite — maintains full context across sessions and minimizes hallucinations through strict document grounding
- Designed a modular, scalable RAG architecture optimized for **local LLM inference**, enabling fast and accurate responses entirely on-device

**Stack:** Python · Ollama · LangChain · LangGraph · FAISS · SQLite · OCR · Sentence Transformers

---

### Medical Chatbot — RAG · LangChain · GPT-4 · Pinecone · AWS · Docker · CI/CD

End-to-end RAG-based chatbot for medical query resolution, grounded in a 637-page medical textbook.

- Built full ingestion pipeline with **Sentence Transformer embeddings** and **Pinecone** vector storage
- Orchestrated **LangChain + GPT-4** for context-aware, retrieval-augmented response generation
- Deployed on **AWS EC2** with Docker containerization and a complete **GitHub Actions CI/CD pipeline**

**Stack:** Python · LangChain · GPT-4 · Pinecone · Sentence Transformers · AWS EC2 · Docker · GitHub Actions

---

### Heart Disease Prediction via Retina Images — YOLO · TensorFlow · OpenCV · Roboflow · CNN

Computer vision system for detecting cardiovascular risk markers from retinal scans.

- Implemented **YOLO** for precise detection and classification of retinal features linked to heart disease
- Annotated **800+ images** using Roboflow, cutting manual preprocessing time by **50%**
- Designed and trained **CNNs** with TensorFlow and OpenCV, improving marker identification accuracy by **25%**

**Stack:** Python · YOLO · TensorFlow · OpenCV · Roboflow · CNN

---

### Industrial Object Detection — YOLO · AWS SageMaker · TensorFlow Lite · Android

Real-time on-device defect detection system for industrial use cases.

- Built full ML pipeline: data annotation → augmentation → training → evaluation on **AWS SageMaker**
- Converted trained YOLO model to **TensorFlow Lite** and integrated into an **offline Android app**
- Achieved smooth, accurate real-time object detection entirely on-device with no internet dependency

**Stack:** Python · YOLO · AWS SageMaker · TensorFlow Lite · Android · Roboflow
